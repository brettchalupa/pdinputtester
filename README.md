# PDInputTester

A simple input testing utility for Playdate.

Useful for testing whether or not a given input functionality is working reliably or not.

Download the latest release from [GitHub](https://github.com/brettchalupa/pdinputtester/releases/latest) or [itch.io](https://brettchalupa.itch.io/pdinputtester).

[Made with Sunny ☀️](https://github.com/brettchalupa/sunny)

[Learn how to make games for Playdate!](https://leanpub.com/playdatebook)

## Developing

### Prerequisites

- [Playdate SDK](https://play.date/dev/)
- [git](https://git-scm.com/)
- [just](https://just.systems/)
- Ruby
- [itch.io butler](https://itch.io/docs/butler/) - if you want to release on itch.io

On macOS you can install all prerequisites via [Homebrew](https://brew.sh/) by
running:

```
brew bundle
```

### Setup

After cloning, initialize the vendored submodules by running:

```
just setup
```

### Reference

- [Playdate SDK docs](https://sdk.play.date)
- [Designing for Playdate](https://help.play.date/developer/designing-for-playdate/) -
  best practices guide
